# Commodity-accounting
| Таблица           | Назначение                                           |
| ----------------- | ---------------------------------------------------- |
| Products          | Справочник товаров (название, артикул, цены, группа) |
| ProductBarcodes   | Штрих-коды товаров (один основной + дополнительные)  |
| Stock             | Остатки на складе (товар + количество)               |
| InventorySessions | Сессии инвентаризации                                |
| InventoryLines    | Сканированные товары в текущей сессии                |
